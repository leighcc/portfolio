Buttons

Simple CSS for buttons. To create a Primary Button, add the btn btn-primary classname to any <a> or <button> element.

A link

  • Fancy larger or smaller buttons? Add .btn-lg or .btn-sm for additional sizes.

  • Five predefined button styles, each serving its own semantic purpose.

  • To mark a button as disabled, add the disabled classname. Alternatively, add the "disabled" attribute directly to your button.

  • To style a button so that it appears "pressed", add the active classname.

  • Incorporating icon fonts is easy. Put the Font Awesome CSS file on your page and use an <i> element within a button element.

Button Groups

Group multiple buttons together on a single line.

  • Instead of applying button sizing classes to every button in a group, just add .btn-group-* to each .btn-group.
